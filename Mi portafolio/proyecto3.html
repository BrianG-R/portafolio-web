Gestión de Inventario

Detalles completos del proyecto de gestión de inventario con CRUD y base de datos.

Descripción

Este proyecto fue desarrollado para gestionar el inventario de productos de manera eficiente mediante un sistema CRUD (Crear, Leer, Actualizar, Eliminar). La aplicación fue construida utilizando Java y se conecta a una base de datos MySQL para almacenar la información. Se trabajó en el entorno de desarrollo NetBeans, aprovechando su integración con bases de datos y herramientas gráficas. El sistema permite registrar productos, editar información, eliminar registros y consultar el stock en tiempo real.

Uno de los principales retos fue implementar una conexión estable y segura entre la aplicación Java y MySQL, así como manejar validaciones y excepciones correctamente para evitar errores durante la manipulación de datos. También se enfocó en una interfaz de usuario clara y funcional usando Swing.

Características

  • Registro y edición de productos con validación de datos
  • Eliminación y consulta de inventario en tiempo real
  • Conexión a base de datos MySQL usando JDBC
  • Interfaz gráfica desarrollada con Java Swing
  • Manejo de errores y excepciones
  • Desarrollado en NetBeans
  • Tecnologías: Java, MySQL, JDBC, Swing
