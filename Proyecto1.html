Detalles completos de SweetScape

Descripción

SweetScape es una aplicación móvil desarrollada con Visual Studio 2022 utilizando .NET y Xamarin, pensada para optimizar la gestión de una pastelería.

Permite registrar clientes, productos y sucursales mediante un sistema CRUD. Se enfocó en una interfaz simple y adaptable a dispositivos móviles.

Tecnologías utilizadas: .NET, Xamarin, C#, HTML, CSS.

Características

  • Registro y gestión de clientes
  • CRUD completo para productos de la pastelería
  • Administración de sucursales
  • Interfaz móvil intuitiva y amigable
  • Desarrollado en Xamarin con Visual Studio 2022
  • Conectividad con bases de datos locales
  • Tecnologías: .NET, Xamarin, C#, HTML, CSS
